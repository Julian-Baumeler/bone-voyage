Get the local engine

This website is only the UI. Heavy CT work runs on a small program on your computer — nothing is uploaded. Download once, start it, then come back here (this page auto-connects).

  1. 1 Download the engine

    Source package from GitHub (Python project + install scripts).

    Download engine (.zip) Open GitHub repo
  2. 2 Install (one command)

    macOS / Linux with Python 3.11+. Unzip is optional if you use the curl installer.

    curl -fsSL … | bash

    Or after unzipping the download:

    cd bone-voyage-main && bash scripts/install-engine.sh
  3. 3 Start the engine

    Leave the terminal open. Default: http://127.0.0.1:8742

    ~/bone-voyage/start-engine.sh
  4. 4 Connect
